# 屏幕注视检测器 - 完整使用指南

## 🚀 快速开始（3 步搞定）

### 第一步：下载依赖库（首次使用）

```bash
cd /Users/shumizu/.openclaw/workspace
bash download-mediapipe-libs.sh
```

这会下载 MediaPipe AI 库到本地，确保稳定运行。

### 第二步：启动服务器

```bash
node start-server.js
```

### 第三步：打开浏览器

浏览器会自动打开 `http://localhost:3000`，或手动访问该地址。

授予摄像头权限即可开始使用！

## 🔧 故障排查

### 问题 1：依赖库加载失败

**症状：** 页面显示"依赖库加载失败"

**解决：**
```bash
# 1. 运行下载脚本
bash download-mediapipe-libs.sh

# 2. 如果下载失败，检查网络
# 暂时关闭代理/VPN
# 尝试使用手机热点

# 3. 刷新页面重试
```

### 问题 2：摄像头无法访问

**症状：** 点击"开始检测"后没有反应或报错

**解决：**
1. **确保通过 localhost 访问**（不要直接双击打开 HTML）
2. **检查浏览器权限设置**：
   - Safari: 设置 → 网站 → 摄像头
   - Chrome: 设置 → 隐私和安全 → 网站设置 → 摄像头
3. **关闭其他使用摄像头的程序**（Zoom、微信、Teams 等）

### 问题 3：端口 3000 被占用

**症状：** `错误：端口 3000 已被占用`

**解决：**
```bash
# 方法 1: 找到并关闭占用程序
lsof -i :3000
kill -9 <PID>

# 方法 2: 修改端口
# 编辑 start-server.js，将 PORT = 3000 改为其他值
```

## ⚙️ 功能说明

### 核心功能
- ✅ AI 人脸识别，检测是否在看屏幕
- ✅ 实时统计专注时间、离开次数、专注率
- ✅ 视线离开提醒（视觉闪烁 + 声音）
- ✅ 番茄钟式休息提醒（可自定义间隔）
- ✅ 性能模式（降低 CPU 占用）

### 可调节设置
| 设置 | 默认值 | 说明 |
|------|--------|------|
| 离开检测阈值 | 3 秒 | 视线离开多久后提醒 |
| 休息提醒间隔 | 25 分钟 | 番茄钟工作时间 |
| 声音提醒 | 开启 | 提醒时播放声音 |
| 性能模式 | 开启 | 跳过部分帧降低 CPU |

## 💡 使用技巧

1. **放在屏幕旁边**：让摄像头能拍到你的脸
2. **调整阈值**：如果太敏感，增加"离开检测阈值"
3. **性能模式**：如果电脑发热，开启性能模式
4. **休息提醒**：配合番茄工作法，每 25 分钟休息 5 分钟
5. **统计数据**：关注"专注率"，逐步提高

## 🆘 常见问题

### Q: 可以直接双击打开 HTML 文件吗？
**A:** Safari 可以，但 Chrome/Edge 不行。推荐始终使用 `node start-server.js` 启动。

### Q: 第一次加载很慢？
**A:** 正常！首次需要下载 AI 模型（约 5-10MB）。下载后会缓存，下次就快了。

### Q: 检测不准确怎么办？
**A:** 
- 确保光线充足
- 正对摄像头
- 调整"离开检测阈值"
- 检查摄像头分辨率（建议 640x480 以上）

### Q: 可以在手机上用吗？
**A:** 理论上可以，但设计初衷是桌面端。手机屏幕小，效果可能不佳。
