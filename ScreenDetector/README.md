# ScreenDetector - iOS 屏幕注视检测应用

一个使用 Vision 框架和 AVFoundation 实现的 iOS 应用，用于实时检测用户是否在看屏幕。

## 📱 功能特性

### 核心功能
- ✅ **实时人脸检测** - 使用 Vision 框架检测用户脸部
- ✅ **眼睛状态分析** - 判断眼睛是否睁开、视线方向
- ✅ **专注状态显示** - 绿色=在看屏幕，红色=视线离开
- ✅ **智能提醒** - 视线离开超过设定时间（默认 3 秒）时发出提醒
- ✅ **专注统计** - 记录专注时间、离开次数、专注率

### UI 特性
- 🎨 简洁的界面设计
- 📹 实时摄像头预览
- 🟢🔴 状态指示灯（绿/红）
- ▶️⏹️ 开始/停止按钮
- 📊 统计面板（专注时间、离开次数、平均专注时长等）
- ⚙️ 设置面板（可调整检测阈值）

### 权限处理
- 🔒 规范请求摄像头权限
- 💡 权限被拒绝时显示友好提示和设置引导

## 🛠️ 技术栈

- **语言**: Swift 5.0+
- **框架**:
  - AVFoundation（摄像头）
  - Vision Framework（人脸/眼睛检测）
  - UIKit（界面）
- **最低支持**: iOS 15.0+
- **部署目标**: iPhone/iPad，或作为 Mac Catalyst 应用在 macOS 上运行

## 🚀 如何编译和运行

### 前提条件
- macOS 12.0+
- Xcode 15.0+
- iOS 15.0+ 设备或模拟器

### 步骤

#### 1. 打开项目

在终端中运行：
```bash
cd /Users/shumizu/.openclaw/workspace/ScreenDetector
open ScreenDetector.xcodeproj
```

或者在 Xcode 中：
- 启动 Xcode
- 选择 `File` > `Open...`
- 导航到 `ScreenDetector.xcodeproj` 并打开

#### 2. 选择运行目标

在 Xcode 工具栏中：
- 点击设备选择器
- 选择您的 iPhone/iPad 设备，或选择一个 iOS 模拟器

**注意**: 人脸检测需要摄像头，建议使用真实设备进行测试。

#### 3. 配置签名

如果是第一次运行：
- 在项目导航器中选择 `ScreenDetector` 项目
- 选择 `ScreenDetector` 目标
- 进入 `Signing & Capabilities` 标签
- 选择您的开发团队（需要 Apple ID）
- Xcode 会自动处理证书和配置文件

#### 4. 编译并运行

- 点击工具栏中的 ▶️ 运行按钮
- 或使用快捷键 `Cmd + R`

#### 5. 授权摄像头权限

首次运行时：
- 应用会请求摄像头权限
- 点击 `允许` 以启用视线检测功能

## 📖 使用说明

### 基本操作

1. **开始检测**: 点击绿色的"开始检测"按钮
2. **停止检测**: 点击红色的"停止检测"按钮
3. **查看状态**: 
   - 🟢 绿色指示灯 = 正在专注看屏幕
   - 🔴 红色指示灯 = 视线已离开
   - ⚪ 灰色指示灯 = 未检测到人脸
4. **查看统计**: 在统计面板查看专注时长、离开次数、专注率等
5. **重置统计**: 点击"重置统计"按钮清空所有数据

### 设置

点击"设置"按钮可以调整：
- **视线离开提醒阈值**: 多少秒后发出提醒（默认 3 秒）

### 最佳实践

1. **设备位置**: 将设备放在正前方，与眼睛平齐
2. **光线条件**: 确保环境光线充足，避免背光
3. **距离**: 保持 30-60cm 的使用距离
4. **角度**: 尽量保持面部正对摄像头

## 🔍 技术实现细节

### 人脸检测
使用 Vision 框架的 `VNDetectFaceRectanglesRequest` 检测人脸位置，使用 `VNDetectFaceLandmarksRequest` 检测面部特征点。

### 眼睛状态分析
通过分析眼睛特征点的垂直高度与水平宽度的比例来判断眼睛是否睁开。

### 视线方向估算
通过鼻尖相对于人脸中心的水平偏移来估算视线方向。

### 性能优化
- 视频帧处理在后台线程进行
- UI 更新在主线程进行
- 使用序列号取消过期的检测请求
- 自动丢弃延迟的视频帧

## ⚠️ 注意事项

1. **隐私**: 应用不会存储或上传任何摄像头画面，所有处理都在本地进行
2. **电量**: 持续使用摄像头会增加电量消耗
3. **准确性**: 检测结果可能受光线、角度、遮挡等因素影响
4. **健康**: 本应用仅供参考，不能替代专业的健康监测设备

## 🐛 故障排除

### 摄像头无法启动
- 检查是否已授权摄像头权限
- 在设置中允许应用访问摄像头
- 确保没有其他应用正在使用摄像头

### 检测不准确
- 调整设备位置和角度
- 改善环境光线
- 在设置中调整检测阈值

### 应用崩溃
- 确保使用 iOS 15.0 或更高版本
- 重启设备后重试
- 重新安装应用
